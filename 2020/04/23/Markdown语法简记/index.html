特殊样式：

加粗：** **或__ __

斜体：* *或_ _

下划线: <u> 下划线</u>

代码: ` `

删除线：~~ 删除线~~

分割线：***或---或____

上下标：^ 或 _

Typora的快捷键

  • 加粗：ctrl + B
  • 标题：ctrl + H
  • 插入链接：ctrl + K
  • 插入图片：ctrl + shift + I
  • 无序列表： ctrl + shift + L
  • 一级标题：ctrl + 1
  • 切换原文和语法： ctrl + /

blog不支持的部分功能
